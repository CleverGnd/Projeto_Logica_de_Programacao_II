<h2 id="Projeto">Projeto</h2>

**Objetivo**: Criar um sistema de controle de estoque. O programa deve forncer as seguintes funcionalidades:

   - Cadastrar produto
   - Consultar produto
   - Listar produtos cadastrados
   - Atualizar cadastro
   - Excluir cadastro
